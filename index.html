Sleepy Cat Alarm 개인정보처리방침

앱 이름: Sleepy Cat Alarm (잠자는 고양이 알람)

개발자: [귀하의 이름 또는 회사명]

최종 수정일: 2025년 1월 20일

1. 개인정보의 수집 및 이용

Sleepy Cat Alarm은 사용자의 개인정보를 수집하지 않습니다.

1.1 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 연락처
  • 사진, 파일 등 미디어 정보

1.2 로컬 저장 데이터

다음 데이터는 사용자의 기기에만 저장되며, 외부로 전송되지 않습니다:

  • 알람 설정 정보 (시간, 라벨, 활성화 상태)
  • 고양이 수면 기록 (수면 시작/종료 시간, 방해 횟수)
  • 앱 설정 및 테마 정보

2. 권한 사용 목적

본 앱은 다음 권한을 사용합니다:

2.1 필수 권한

  • 알림 (NOTIFICATIONS): 설정한 시간에 알람 알림을 표시하기 위해 사용
  • 전체 화면 알림 (USE_FULL_SCREEN_INTENT): 알람 시간에 전체 화면 알람을 표시하기 위해 사용
  • 화면 켜기 (WAKE_LOCK): 알람 발동 시 화면을 자동으로 켜기 위해 사용
  • 진동 (VIBRATE): 알람 발동 시 진동 기능을 제공하기 위해 사용

2.2 선택 권한

  • 현재 없음

3. 데이터 저장 및 보안

  • 모든 데이터는 사용자의 기기 내부에만 저장됩니다 (AsyncStorage 사용)
  • 개발자는 사용자의 데이터에 접근할 수 없습니다
  • 앱 삭제 시 모든 데이터가 함께 삭제됩니다
  • 외부 서버로 데이터를 전송하지 않습니다

4. 제3자 제공 및 공유

본 앱은 사용자의 개인정보를 제3자에게 제공하거나 공유하지 않습니다.

4.1 사용하지 않는 서비스

  • 광고 서비스
  • 분석 도구 (Google Analytics 등)
  • 소셜 미디어 연동
  • 클라우드 백업

5. 아동의 개인정보 보호

본 앱은 만 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 본 앱은 개인정보를 수집하지 않으므로, 모든 연령대의 사용자가 안전하게 사용할 수 있습니다.

6. 사용자의 권리

사용자는 다음 권리를 가집니다:

  • 앱 내 설정에서 알람 데이터 삭제
  • 앱 삭제를 통한 모든 데이터 제거
  • 언제든지 알림 권한 거부 (설정 > 앱 > Sleepy Cat Alarm > 권한)

7. 개인정보처리방침 변경

본 개인정보처리방침은 법령 및 서비스 변경에 따라 수정될 수 있습니다. 변경 시 앱 업데이트 공지 또는 본 페이지를 통해 안내됩니다.
